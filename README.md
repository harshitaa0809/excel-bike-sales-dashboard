# Excel Bike Sales Dashboard

## Project Overview

This project involves analyzing bike sales data using Microsoft Excel and developing an interactive dashboard to visualize customer purchasing behavior and key business trends.

The analysis focuses on understanding how demographic and lifestyle factors influence bike purchase decisions and presents the findings through a user-friendly dashboard.

## Objectives

- Analyze customer demographics and purchasing patterns
- Identify factors influencing bike purchases
- Create interactive visualizations for data-driven insights
- Develop a professional dashboard for reporting purposes

## Project Workflow

### Data Preparation
- Reviewed and cleaned the dataset
- Verified data consistency and accuracy
- Prepared the dataset for analysis

### Data Analysis
- Performed analysis using Pivot Tables
- Examined customer income, age, occupation, and commuting patterns
- Compared purchase behavior across different customer segments

### Dashboard Development
- Created Pivot Charts to visualize insights
- Added Slicers for interactive filtering
- Designed an intuitive dashboard layout for easy interpretation

## Key Insights

- Relationship between customer income and bike purchases
- Purchase trends across different age groups
- Impact of commute distance on buying decisions
- Regional variations in bike purchases
- Customer demographic patterns influencing sales

## Tools & Technologies

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design

## Outcome

The final dashboard provides a clear and interactive view of bike sales trends, enabling users to explore customer behavior and identify key factors affecting purchasing decisions.
